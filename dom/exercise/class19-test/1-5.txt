1. DOM은 어떤 말의 약자인가요?
    (1. Document Object Model))

2. HTML 문서가 완전히 로드된 후에 (DOM이 완전히 구성된 후에) 자바 스크립트 코드를 실행할 때 가장 적잘한 이벤트는 무엇인가?
    (2. DOMContentLoaded)

3. 자바 스크립트로 스타일을 변경하려면 어떤 속성을 사용해야 하나요?
    (1. 요소.style)

4. <ul> 요소 내부에 새로운 <li> 요소를 추가할 때 자바 스크립트 에서 사용하는 메서드는 무엇인가요?
    (1.appendChild)

5. DOM 에서 요소의 텍스트 내용을 변경할 때 사용할 수 없는 속성을 무엇인가요? 
    (3 . textHTML)
